About Database Triggers

Database triggers are automated procedures that run when specific events occur in the database.

Active Triggers in This Database:
  • tr_alumni_before_insert - Validates alumni data
  • tr_student_before_insert - Validates student data
  • tr_feedback_before_insert - Validates feedback
  • tr_feedback_after_insert - Auto-logs feedback
  • tr_mentorship_session_after_insert - Creates mentor relationships
Test Results Guide
